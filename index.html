Grammar reference
Core patterns — units 9–16
-고 싶다
I want to V
Attach to verb stem. 1st/2nd person only.
3rd person → -고 싶어하다
저는 비빔밥을 먹고 싶어요.I want to eat bibimbap.
-(으)ㄹ 수 있다/없다
can / cannot V
Consonant stem → -을 수 있다
Vowel stem → -ㄹ 수 있다
기타를 칠 수 있어요.I can play guitar.
-(으)려고 하다
plan / intend to V
Consonant stem → -으려고 하다
Vowel stem → -려고 하다
제주도에 가려고 해요.I'm planning to go to Jeju.
-아/어서
because / and then
-아서 after ㅏ/ㅗ, -어서 elsewhere
하다 → 해서. No independent tense.
날씨가 좋아서 공원에 가요.The weather's nice so I'm going to the park.
-(으)니까
because / since
After consonant → -으니까
After vowel → -니까
Preferred with commands & proposals
비가 오니까 우산을 가지고 가세요.It's raining so take an umbrella.
-는 것을 좋아하다
to like V-ing
Attach -는 것 to verb stem → noun
Then add 을/를 좋아하다
저는 요리하는 것을 좋아해요.I like cooking.
ㅂ 불규칙
ㅂ irregular
ㅂ → 우 before vowel endings (+어요 → 워요)
덥다 → 더워요 · 춥다 → 추워요hot / cold
르 불규칙
르 irregular
Drop ㅡ, double the ㄹ before -아/어
부르다 → 불러요 · 모르다 → 몰라요to sing / to not know
